# Data-Structure-using-Java
 Learn Data Structures & Algorithms in Easy Way by creating custom collection classes in Java.
## Topic covers
  * Array
  * ArrayList
  * Stack
  * Queue
  * Searching and sorting
  * Linked List
      * SL List
      * DL List
  * Tree
      * BST Tree
      * AVL Tree
  * Graph
  * Hashing 
      * HashTable
      * HashSet
      * HashMap
